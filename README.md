# SRL Analytics Platform

An advanced real-time Self-Regulated Learning (SRL) analytics platform upgraded from a standard LMS.

## Features
- **Real-time Event Pipeline**: Streaming telemetry for student activity.
- **Advanced Video Analytics**: Confusion detection, rewatch heatmaps, and interactive checkpoints.
- **SRL Modeling**: Dynamic scoring of student learning strategies (Planning, Monitoring, Control, Reflection).
- **Teacher Dashboards**: Class-wide strategic gaps and automated student behavior clustering.
- **Intervention System**: Teacher-assigned adaptive support materials.

## How to Execute

### 1. Clone the repository
```bash
git clone https://github.com/geethatigulla/srl.git
cd srl
```

### 2. Install dependencies
Make sure you have [Node.js](https://nodejs.org/) installed.
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```
The application will be available at `http://localhost:5173`.

### 4. Default Credentials
- **Student**: `student@edu.com` / `password`
- **Teacher**: `teacher@edu.com` / `password`
